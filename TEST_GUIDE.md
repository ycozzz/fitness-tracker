2. 選擇餐別（早餐/午餐/晚餐/小食）

## 🔧 開發者測試

### API 測試

#### 1. 上傳圖片 API
```bash
curl -X POST http://192.168.131.21:3001/api/analyze \
  -F "image=@test.jpg"
```

#### 2. 儲存餐食 API
```bash
curl -X POST http://192.168.131.21:3001/api/meals \
  -H "Content-Type: application/json" \
  -d '{
    "mealType": "lunch",
    "foodName": "雞胸肉",
    "calories": 450,
    "protein": 35,
    "carbs": 52,
    "fat": 8
  }'
```

#### 3. 查詢今日餐食
```bash
curl "http://192.168.131.21:3001/api/meals?date=2026-03-05"
```

### 數據庫測試

```bash
# 連接數據庫
sqlite3 /home/user01/fitness-tracker/fitness.db

# 查看所有表
.tables

# 查看餐食記錄
SELECT * FROM meals ORDER BY created_at DESC LIMIT 10;

# 查看用戶資料
SELECT * FROM profiles WHERE user_id=1;

# 統計今日卡路里
SELECT 
  meal_type,
  COUNT(*) as count,
  SUM(calories) as total_calories,
  SUM(protein) as total_protein
FROM meals 
WHERE user_id=1 AND meal_date=date('now')
GROUP BY meal_type;
```
